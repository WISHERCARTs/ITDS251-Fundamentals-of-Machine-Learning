# Lab 6 — Deep Learning Regression

ITDS251 Fundamentals of Machine Learning

## Overview

This lab explores **Deep Learning for Regression** using neural networks with Keras/TensorFlow on the **California Housing** dataset.

### Topics Covered

- Single-node neural network (perceptron)
- Multi-node neural network
- Hidden layers & deep neural networks
- Model evaluation with MSE / MAE / R²
